# HFS
文件服务器

## Usage
```
Usage of ./hfs [arguments]:

-cert string
    https cert file
-dir string
    http file directory (default "./")
-host string
    ip address (default "0.0.0.0")
-key string
    https key file
-port string
    listen http port (default "8910")
-prefix string
    http route path (default "/")
-tls string
    enable https (default "false")
-v string
    release version
```
